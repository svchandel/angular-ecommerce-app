# E-Commerce Store - Angular Application


##  Installation & Setup

1. Clone the Repository
```bash
git clone <repository-url>
cd ecommerce-app
```

2. Install Dependencies
```bash
npm install
```

3. Start the Development Server
```bash
ng serve
```

The application will be available at `http://localhost:4200/`
